## 📌 Project Overview

Eiseb Country Traders is a cattle baron organisation that currently relies on a manual filing system to manage their financial operations. This project delivers a **web-based Livestock Financial Management System** to automate and digitise the financial processes related to livestock management.

### Module Assigned
**Livestock Financial Management System** — tracking income, expenses, livestock valuations, transaction records, and financial reporting for the farm.

---

## 🎯 Key Functionalities

- Livestock asset valuation and recording
- Income tracking (livestock sales, auction proceeds)
- Expense tracking (feed, veterinary, transport costs)
- Financial reports and summaries
- User registration and login
- Contact Us page

## 🚀 Getting Started

### Prerequisites
- JDK 11+
- Apache Tomcat 9+
- MySQL 8+
- Any IDE (Eclipse / IntelliJ / VS Code)

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR-USERNAME/eiseb-financial.git
   cd eiseb-financial
   ```

2. **Set up the database**
   ```bash
   mysql -u root -p < database/schema.sql
   ```

3. **Deploy to Tomcat**
   - Copy the project to Tomcat's `webapps/` folder
   - Start Tomcat and navigate to `http://localhost:8080/eiseb-financial`
